# CLAUDE.md

Web design agency system for cloning and customizing websites for clients.

Rules are split into focused files in `.claude/rules/`:

| Rule | When it applies |
|---|---|
| `project-structure.md` | Always — repo layout and client folder conventions |
| `stack-decisions.md` | When starting a new project or choosing a framework |
| `dev-commands.md` | When working with `package.json` / npm projects |
| `design-workflow.md` | When cloning or recreating a website design |
| `html-css-conventions.md` | When writing any `.html` or `.css` file |
