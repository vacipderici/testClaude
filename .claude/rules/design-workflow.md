Follow this workflow when recreating a website design for a client:

1. Analyze the reference website's layout, color palette, typography, and component structure
2. Extract CSS tokens: colors, spacing scale, font families, border-radius, shadows, animations
3. Recreate the structure with clean, semantic HTML
4. Apply custom branding: swap colors, fonts, logo, and copy for the client
5. Replace all placeholder content with client-specific content
6. Ensure responsiveness across mobile (≤767px), tablet (≤1024px), and desktop

Use the `clone-site` skill for automated CSS extraction and recreation.
