Quick Start

Get up and running with Mocha in under five minutes. This guide walks you through installing the CLI, authenticating your account, creating your first agent, and running it — all from your terminal.

Before you begin, make sure you have Node.js 18+ installed. Mocha agents run in any Node.js environment and deploy seamlessly to Mocha Cloud or your own infrastructure.

1 Step 1

Install Mocha CLI

The Mocha CLI is the primary way to create, manage, and deploy agents. Install it globally via npm:

bash 
npm install -g @mocha/cli

After installation, verify it's working by running mocha --version. You should see the current CLI version printed to your terminal.

2 Step 2

Authenticate

Connect the CLI to your Mocha account. This stores a secure token in your local config so you don't need to log in again on subsequent commands.

bash 
mocha auth login

A browser window will open prompting you to sign in. Once authenticated, you'll see a confirmation in your terminal. If you're in a headless environment, use mocha auth login --token <YOUR_API_KEY> instead.

3 Step 3

Create your first agent

Create a new file called my-agent.json with the following configuration. This defines a simple agent with a name, model, system prompt, and the tools it can use.

json 
{
  "name": "my-agent",
  "version": "1.0.0",
  "model": "mocha-3-sonnet",
  "system": "You are a helpful assistant. Answer questions clearly and concisely.",
  "tools": ["web_search", "code_interpreter", "file_read"],
  "memory": {
    "enabled": true,
    "strategy": "sliding_window",
    "maxTokens": 8192
  },
  "config": {
    "temperature": 0.7,
    "maxSteps": 20,
    "timeout": 30000
  }
}

The tools array specifies which built-in Mocha tools your agent can access. The memory block enables persistent context across turns using a sliding-window strategy.

4 Step 4

Run your agent

With your config saved, start your agent interactively from the command line. Mocha will load your configuration, initialize the context engine, and open a REPL session:

bash 
mocha run my-agent

# Expected output:
# ✓ Loaded agent config: my-agent v1.0.0
# ✓ Context engine initialized (sliding_window, 8192 tokens)
# ✓ Tools registered: web_search, code_interpreter, file_read
# ● Agent ready. Type your message below or Ctrl+C to exit.
# >

You can also run your agent non-interactively by passing a prompt directly: mocha run my-agent --prompt "Summarize the latest AI news"
